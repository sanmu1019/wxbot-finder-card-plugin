# 接口契约

## Finder 短链网关

### 请求

```text
POST /api/v1/finder/share-url
```

请求头：

```text
X-API-Key: YOUR_FINDER_KEY
Content-Type: application/json
```

请求体：

```json
{
  "object_id": "OBJECT_ID",
  "object_nonce_id": "OBJECT_NONCE_ID",
  "scene": 40
}
```

独立插件使用 POST，不把 nonce 放在 URL 查询参数中。

### 成功响应

```json
{
  "code": 0,
  "msg": "ok",
  "data": {
    "object_id": "OBJECT_ID",
    "sph_url": "https://weixin.qq.com/sph/SHORT_CODE",
    "feedH5Url": "https://weixin.qq.com/sph/SHORT_CODE",
    "scene": 40
  }
}
```

插件按顺序读取：

```text
data.sph_url
data.feedH5Url
data.share_url
```

### 常见错误

| HTTP | 含义 |
|---|---|
| 401 | Finder API Key 错误 |
| 422 | object ID、nonce 或请求格式错误 |
| 503 | Chromium 未登录或短链模块未就绪 |
| 502 | 管理后台未返回有效短链 |

## wxsph-api

### 请求

```text
GET /api/wxsph?url=<URL_ENCODED_SPH>
```

请求头：

```text
X-API-Key: YOUR_WXSPH_KEY
```

### 成功响应

插件兼容以下结构：

```json
{
  "code": 200,
  "msg": "解析成功",
  "data": {
    "title": "标题",
    "desc": "简介",
    "cover": "https://example.invalid/cover.jpg",
    "url": "https://example.invalid/video.mp4",
    "quality": "1080p",
    "author": {
      "name": "作者"
    },
    "video_backup": [
      {
        "url": "https://example.invalid/video-backup.mp4"
      }
    ]
  }
}
```

`code` 可以是数值或字符串形式的 `0/200`。

### 说明

当前 `wxsph-api` 实现依赖元宝会话。Cookie 过期会导致解析失败，但不会影响
Finder 网关生成短链。

## protocol-core 可选接口

### 请求

```text
POST /Finder/GetCommentDetail
Content-Type: application/x-www-form-urlencoded
```

表单：

```text
wxid=BOT_WXID
objectId=OBJECT_ID
objectNonceId=OBJECT_NONCE_ID
cgi=3906
```

该接口应只监听本机或受保护的管理网络。

### 响应要求

插件接受：

```json
{
  "Code": 0,
  "Success": true,
  "Data": {
    "title": "标题",
    "nickname": "作者",
    "media": [
      {
        "full_url": "https://example.invalid/media",
        "decode_key": "EXAMPLE_KEY"
      }
    ]
  }
}
```

字段名同时兼容部分大小写变体：

```text
full_url / fullUrl / FullURL
decode_key / decodeKey / decrypt_key / decryptKey
```

如果只返回基础 URL，则还需要：

```text
url + url_token
```

插件会拼成完整 URL。

## 超时建议

```text
Finder 短链网关：30 秒
wxsph-api：18 秒，最多重试 2 次
protocol-core：15 秒
视频下载：至少 30 秒
```

## 安全要求

- API Key 放在请求头，不放 URL。
- Finder 网关使用 POST。
- 不记录完整 nonce。
- 不记录完整带 token 的媒体 URL。
- protocol-core 不对公网开放。
- 跨机器调用使用 HTTPS。
